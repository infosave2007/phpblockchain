Blockchain API

RESTful API for blockchain interaction

API Documentation
The API endpoints are being implemented. Available endpoints will include:
Blockchain Endpoints
GET /api/blocks
Get all blocks
GET /api/blocks/{id}
Get specific block
GET /api/transactions
Get all transactions
Wallet Endpoints
POST /api/wallet/create
Create new wallet
GET /api/wallet/{address}
Get wallet balance
POST /api/transaction/send
Send transaction
